MathJax Equation Tags

Equations can get reference numbers using the \tag{} macro, e.g:

\[E=mc^2\tag{1}\]
produces \[E=mc^2\tag{1}\]

Note that the contents of \tag{} are in text mode, so if you want to include math symbols, use \tag{$...$}. E.g.,

\[E=mc^2\tag{$\bullet$}\]
produces \[E=mc^2\tag{$\bullet$}\]
MathJax Equation References

Equations tags can be referenced using the \label{} and \ref{} or \eqref{} macros.

For example, 
The famous equation (\ref{Einstein}) by Einstein, ...
\[E=mc^2\label{Einstein}\tag{1}\]
produces
The famous equation (\ref{Einstein}) by Einstein, ...
\[E=mc^2\label{Einstein}\tag{1}\]
MathJax Equation Numbering

Displayed equations can be numbered automatically.
  • Numbering is off by default
  • Numbering can follow AMS LATEX conventions
    (starred and unstarred environments)
  • Every displayed equation can be numbered

For example, 
MathJax.Hub.Config({
  TeX: {equationNumbers: {autoNumber: "AMS"}}
});

Other autoNumber values are "none" or "all".
