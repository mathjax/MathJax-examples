Colors, Backgrounds, and Borders

  • Expressions can be colored using the \color{} macro.
  • For example
    \color{red}{R} + \color{green}{G} + \color{blue}{B}
    produces \[\color{red}{R} + \color{green}{G} + \color{blue}{B}\]

  • A more LATEX-compatible \color command can be obtained from the MathJax color extension.

    (See the MathJax Documentation for details.)

  • Background colors and borders can be added using the \bbox{} macro.
  • For example 
    \bbox[#FD9,border:2px solid purple,10px]{
  \bbox[red]{r} +
  \bbox[green,5px]{\color{yellow}{g}} +
  \bbox[border:2px dashed blue,3px]{b}
}
    produces \[ \bbox[#FD9,border:2px solid purple,10px]{ \bbox[red]{r} + \bbox[green,5px]{\color{yellow}{g}} + \bbox[border:2px dashed blue,3px]{b} } \]

  • Background color, padding amount, and border specifications can be in any order.

Expressions can be "enclosed" in various ways via the \enclose{} macro.

For example 
\enclose{box}{
  \enclose{circle}{\bbox[5px]{x+y}} +
  \enclose{circle}{\enclose{circle}{z}} -
  \enclose{updiagonalstrike,downdiagonalstrike}{w}
}
produces \[ \enclose{box}{ \enclose{circle}{\bbox[5px]{x+y}} + \enclose{circle}{\enclose{circle}{z}} - \enclose{updiagonalstrike,downdiagonalstrike}{w} } \]

You can combine \color{} with \enclose{}.

For example 
\color{red}{
  \enclose{updiagonalstrike,downdiagonalstrike}{
    \color{black}{x+y}
  }
}
produces \[ \color{red}{ \enclose{updiagonalstrike,downdiagonalstrike}{ \color{black}{x+y} } } \]
  • The possible enclosings are:
    box, roundedbox, circle, left, right, top, bottom, verticalstrike, horizontalstrike, updiagonalstrike, downdiagonalstrike, actuarial, longdiv, radical, and madrawb.
  • The \enclose{} macro is defined in an extension that you must load explicitly if you need it.
  • Include the following in your configuration: 
    MathJax.Hub.Config({
  TeX: {extensions: ["enclose.js"]}
});
  • See the MathJax Documentation for details.
